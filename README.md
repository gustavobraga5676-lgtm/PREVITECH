# PREVITECH

Sistema de gerenciamento de chamados técnicos desenvolvido para a ExpoTech 2026.

---

## Objetivo

O PREVITECH foi desenvolvido para auxiliar empresas no controle de chamados técnicos, manutenção de equipamentos e gerenciamento administrativo.

---

## Funcionalidades

- Login de usuários
- Cadastro de usuários
- Dashboard interativo
- Abertura de chamados
- Acompanhamento técnico
- Controle de manutenção
- Controle de estoque
- Gerenciamento administrativo

---
